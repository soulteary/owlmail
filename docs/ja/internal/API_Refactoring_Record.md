## 概要

このドキュメントは、OwlMail の API リファクタリングプロセスを記録し、MailDev 互換の API エンドポイントから新しい RESTful API 設計（`/api/v1/`）への移行を文書化しています。リファクタリングは、改善された API 設計パターンを導入しながら、完全な後方互換性を維持します。

> **注**：翻訳が完了すると、完全なコンテンツが利用可能になります。それまでの間、完全な詳細については英語版を参照してください。
