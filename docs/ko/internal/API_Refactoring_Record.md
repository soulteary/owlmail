## 개요

이 문서는 OwlMail의 API 리팩토링 프로세스를 기록하며, MailDev 호환 API 엔드포인트에서 새로운 RESTful API 설계(`/api/v1/`)로의 마이그레이션을 문서화합니다. 리팩토링은 개선된 API 설계 패턴을 도입하면서 완전한 하위 호환성을 유지합니다.

> **참고**: 번역이 완료되면 전체 내용을 사용할 수 있습니다. 그동안 전체 세부사항은 영어 버전을 참조하세요.
