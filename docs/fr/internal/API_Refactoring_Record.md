## Aperçu

Ce document enregistre le processus de refactorisation de l'API d'OwlMail, documentant la migration des points de terminaison API compatibles avec MailDev vers une nouvelle conception d'API RESTful (`/api/v1/`). La refactorisation maintient une compatibilité arrière complète tout en introduisant des modèles de conception d'API améliorés.
